For the first time to run the code:

1. Go to cmd and go to mealmate/miniIT file location

2. run 'pip install Django' to install django

3. run 'pip install django-crispy-forms'

4. run 'pip install django-filter'

5. run 'pip install django-import-export'

6. run 'python manage.py runserver'

7. get the link from cmd or In your browser, go to http://localhost:8000

To log in to admin page:
1. go to http://localhost:8000/admin
2. Username : mealmate   password : mm123456789
